# Fake Signature Detection System

A machine learning based web application that detects whether a signature is genuine or fake.

## 📌 Features

- Upload signature image
- Detect Genuine Signature
- Detect Fake Signature
- Simple and user-friendly web interface
- Built using Machine Learning

## 🛠 Technologies Used

- Python
- Flask
- OpenCV
- Scikit-learn
- HTML
- CSS

## 📂 Project Structure

fake-signature-detection/
│── app.py
│── train.py
│── model.pkl
│── templates/
│   └── index.html
│── static/
│   └── style.css

## ▶️ How to Run Project

1. Install required libraries:

pip install flask opencv-python scikit-learn joblib

2. Run application:

python app.py

3. Open browser:

http://127.0.0.1:5000

## 🎯 Output

- Genuine Signature ✅
- Fake Signature ❌
